# CDX - Clock Divider/Multiplier

Clock divider and multiplier firmware for Uncertainty.

## Outputs

| Gate | Function | Description |
|------|----------|-------------|
| 1 | /2 | Half speed |
| 2 | /4 | Quarter speed |
| 3 | /8 | Eighth speed |
| 4 | /12 | Dotted notes/triplets |
| 5 | /16 | Sixteenth speed |
| 6 | x2 | Double speed |
| 7 | x3 | Triple speed (triplets) |
| 8 | x4 | Quadruple speed |
